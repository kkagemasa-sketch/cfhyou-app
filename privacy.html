🔒 プライバシーポリシー

Housing FP株式会社は、HouseFitをご利用いただくユーザーのプライバシーを尊重し、個人情報の保護に努めます。

本プライバシーポリシー（以下「本ポリシー」）は、Housing FP株式会社（以下「当社」）が、HouseFit（以下「本サービス」）の提供にあたり取得する個人情報の取扱方針を定めるものです。

第1条取得する個人情報

当社は、本サービスの提供にあたり、以下の個人情報を取得することがあります。

情報の種類具体例取得方法
登録情報氏名、メールアドレス、所属法人名、電話番号ユーザー登録時
決済情報クレジットカード情報（決済代行業者経由）決済時
顧客情報（ユーザーが入力）顧客の氏名、年齢、年収、家族構成、保有資産、ローン情報等本サービス使用時
利用ログIPアドレス、ブラウザ情報、操作履歴自動取得
エラーログJavaScriptエラー情報、発生したコード位置、ブラウザ環境（個人情報・顧客情報は除外）自動取得（Sentry経由 or ローカル保存）
Cookie等セッション情報、認証トークン自動取得

第2条個人情報の利用目的

当社は、取得した個人情報を以下の目的で利用します。

  1. 本サービスの提供・運営・改善のため
  2. ユーザーからのお問い合わせへの対応
  3. 利用料金の請求・決済処理
  4. 本サービスに関する重要なお知らせ、メンテナンス情報の通知
  5. 新機能、キャンペーン、関連サービスのご案内（ユーザーの同意を得た場合）
  6. 利用規約違反・不正利用の調査・対応
  7. 本サービスの品質向上のための統計分析（個人を特定しない形）
  8. 法令上の義務を履行するため

第3条個人情報の保存場所

当社が取得する個人情報の保存場所は以下のとおりです。

  • 登録情報・決済情報: 当社が管理するサーバーおよびFirebase（Google Cloud Platform、データセンターは原則として米国）
  • 顧客情報（ユーザー入力データ）: 原則としてユーザーのブラウザ内（IndexedDB／localStorage）に保存され、当社サーバーには送信されません。ただし、ユーザーが意図的にクラウド保存機能を利用した場合は当社サーバーに保存されます。
  • 利用ログ・Cookie: 当社サーバーおよび解析サービス（導入時）

第4条第三者提供

  1. 当社は、以下の場合を除き、ユーザーの同意なく個人情報を第三者に提供しません。
    • 法令に基づく場合
    • 人の生命、身体または財産の保護のために必要がある場合
    • 国の機関等が法令の定める事務を遂行することに対して協力する必要がある場合
    • 事業承継（合併、会社分割、事業譲渡等）に伴い個人情報を提供する場合
  2. 当社は、本サービスの提供のため、以下の業務委託先に個人情報の取扱を委託する場合があります。
    • クラウドインフラ提供事業者（Google Cloud / Amazon Web Services 等）
    • 決済代行事業者（Stripe 等）
    • メール配信事業者
    • 分析・カスタマーサポートツール提供事業者
    • エラー監視サービス（Sentry / Functional Software, Inc.、データセンターは米国）
  3. 業務委託先には、契約により適切な個人情報保護を義務付けます。

第5条顧客情報の取扱いに関するユーザーの責任

  1. 本サービスにおいて、ユーザーは自身の顧客に関する個人情報を入力することができます。
  2. ユーザーは、顧客情報を本サービスに入力する場合、当該顧客本人から個人情報の利用目的について同意を得る責任を負います。
  3. 顧客情報は、ユーザーのブラウザ内に保存されることを原則とし、当社は顧客情報の内容を閲覧しません。

第6条個人情報の安全管理

  1. 当社は、取得した個人情報の漏えい、滅失、毀損の防止その他の安全管理のため、必要かつ適切な措置を講じます。
  2. 具体的な措置:
    • 通信の暗号化（HTTPS）
    • パスワードのハッシュ化保存
    • アクセス権限の最小化
    • 不正アクセス監視
    • 定期的なセキュリティアップデート
    • 従業員への個人情報保護研修

第7条個人情報の保存期間

  • 登録情報: アカウント有効期間中、および退会後 [3ヶ月] までは保持し、その後速やかに削除します。
  • 決済情報: 法令に基づく保存期間（取引記録は7年間）に従います。クレジットカード番号自体は当社に保存しません。
  • 顧客情報: ユーザーのブラウザ内に保存される情報は、ユーザーが削除するまで保持されます。
  • 利用ログ: 取得から1年間を目安に保持し、その後削除または匿名化します。

第8条Cookie等の利用

  1. 本サービスでは、ユーザー体験向上、認証維持、利用状況分析のためにCookie等の技術を利用します。
  2. ブラウザ設定によりCookieを無効化することができますが、その場合、本サービスの一部機能（ログイン状態の維持等）が利用できなくなる可能性があります。
  3. 本サービスで利用するCookie:
    • 必須Cookie: 認証、セッション維持
    • 機能Cookie: ユーザー設定の保存
    • 解析Cookie: 利用状況の分析（導入時、ユーザー同意取得）

第9条ユーザーの権利

ユーザーは、当社が保有する自己の個人情報について、以下の権利を有します。

  • 開示請求: 当社が保有する自己の個人情報の開示を求めることができます
  • 訂正・追加・削除: 内容が事実と異なる場合、訂正等を求めることができます
  • 利用停止: 不適切な利用がある場合、利用停止を求めることができます
  • 消去（削除）: 利用目的の達成等により不要となった場合、消去を求めることができます
  • 第三者提供の停止: 第三者への提供停止を求めることができます
  • データポータビリティ: 自己のデータをJSON形式等で出力することができます

請求は、第13条のお問い合わせ窓口までご連絡ください。本人確認の上、法令に従い遅滞なく対応します。

第10条未成年者の個人情報

本サービスは、原則として未成年者の登録を想定していません。未成年者の方が本サービスを利用される場合は、保護者の同意を得てください。

第11条国外への移転

当社が利用する一部のクラウドサービス（Firebase、決済代行等）は、データを日本国外（米国等）のサーバーで処理する場合があります。これらのサービス提供者は、当社所在国と同等以上のデータ保護水準を確保していることを確認のうえ利用しています。

第12条プライバシーポリシーの変更

  1. 当社は、法令の変更や事業内容の変更等に伴い、本ポリシーを変更することがあります。
  2. 重要な変更がある場合は、本サービス上での告知またはメール等でユーザーに通知します。
  3. 変更後の本ポリシーは、本サービス上に掲載した時点から効力を生じます。
