## 1. 目的と背景

### 現状（v1）の課題
- 既存の万が一シミュレーションは**死亡シナリオのみ**
- 障害・就労不能・重い病気といった「生存時のリスク」が未対応
- FP相談で「保険の必要性を数字で示せない」という場面がある

### 本機能（v2）で解決したいこと
- **死亡・高度障害・就労不能・三大疾病**の4シナリオを統合
- 公的保障（遺族年金・障害年金・傷病手当金・高額療養費）を自動計算
- 「公的保障でいくら、不足いくら」を可視化
- 保険の必要保障額の根拠を数字で提示
- お客様への説明ツールとしても機能（症状例・統計値をポップアップ）

### 想定ユーザー
- FP（ご自身）が相談現場で使用
- お客様への提案ツールとして画面を見せながら説明
- 30〜50代の働き盛り世帯がメインターゲット

## 2. 全体方針

### UI/UX方針
- **タブを増やさない**：既存「万が一」タブ内で完結
- **ごちゃごちゃにしない**：アコーディオンで必要な項目のみ表示
- **既存と統一感**：リアルタイム更新・ポップアップは既存パターンを踏襲

### 技術方針
- **既存コードに影響を出さない**：別ブランチで長期開発、完成後統合
- **計算ロジックは共通化**：4シナリオで使い回せる部品を先に作成
- **既存CF表の仕組みを流用**：MR（万一用CF）オブジェクトパターンを拡張

### リスク管理方針
- メンバーが使っている main ブランチは開発中も無傷
- 段階的マージも検討可（完成部分から先に統合）
- 復元用タグをブランチ作成時に打つ

## 3. 対象シナリオ

### 4シナリオの定義

| # | シナリオ | 期間 | 本人収入 | 主な公的保障 | 主な保険 |
|---|---|---|---|---|---|
| 1 | **死亡** | 永続 | ゼロ | 遺族基礎・遺族厚生年金 | 生命保険（死亡） |
| 2 | **高度障害** | 永続（生涯） | ゼロ確定 | 障害年金1級、特別障害者手当 | 生命保険（高度障害特約） |
| 3 | **就労不能** | 永続（回復なし） | 減額（割合 or 手入力） | 障害年金（等級選択）、傷病手当金 | 就業不能保険 |
| 4 | **三大疾病** | 一時的 | 休職中のみ減額 | 高額療養費、傷病手当金 | がん保険・三大疾病保険 |

### シナリオの選択
- **複数選択可**（チェックボックス形式）
- 選択したシナリオだけ入力項目とCF表が生成される
- サマリー表では選択した全シナリオを並列表示

### 高度障害と就労不能の線引き
- **高度障害**：障害年金1級相当、常時介護必要、本人収入ゼロ確定（固定前提）
- **就労不能**：等級1/2/3から選択、収入減の割合を指定（柔軟）
- → お客様が「生命保険」と「就業不能保険」を別々に検討するため、シナリオも分離

## 5. 入力項目

### 5.1 共通入力項目（全シナリオ）

| 項目 | 型 | 必須 | 備考 |
|---|---|---|---|
| 対象者 | 選択（主/副） | ✅ | 誰に発生するか |
| 発生年齢 | 数値 | ✅ | 何歳で発生するか |

### 5.2 死亡シナリオ

| 項目 | 型 | 必須 | 備考 |
|---|---|---|---|
| 生命保険金 | 金額 | 任意 | 死亡時に受取る一時金 |

※既存 v1 ロジックを流用
※遺族年金は自動計算

### 5.3 高度障害シナリオ

| 項目 | 型 | 必須 | 備考 |
|---|---|---|---|
| 高度障害保険金 | 金額 | 任意 | 生命保険の高度障害特約 |
| 介護費用（月額） | 金額 | 任意 | デフォルト：月10万円 |
| 介護形態 | 選択 | ✅ | 在宅／施設 |

※本人収入は自動でゼロに設定
※障害年金1級が自動適用

### 5.4 就労不能シナリオ

| 項目 | 型 | 必須 | 備考 |
|---|---|---|---|
| 障害等級 | 選択（1/2/3級） | ✅ | ポップアップで等級説明 |
| 収入減の指定 | 選択（割合/金額） | ✅ | 2択 |
| 収入減の数値 | 数値 | ✅ | 割合なら%、金額なら万円/年 |
| 就業不能保険金（一時金） | 金額 | 任意 | 一時金として受取 |
| 就業不能保険（月額給付） | 金額 | 任意 | 月次給付 |

※回復はなし（生涯継続）
※障害年金が等級別に自動適用
※傷病手当金は発症後1年6ヶ月のみ適用

### 5.5 三大疾病シナリオ

| 項目 | 型 | 必須 | 備考 |
|---|---|---|---|
| 疾病種別 | 選択 | ✅ | がん／脳卒中／心筋梗塞／その他 |
| 休職期間 | 数値 | ✅ | 月数（復職時に収入復帰） |
| 自己負担医療費 | 金額 | 任意 | 高額療養費適用後 |
| 先進医療費 | 金額 | 任意 | 0〜300万 |
| 疾病保険一時金 | 金額 | 任意 | がん保険・三大疾病保険 |

※休職期間中のみ収入減
※回復後は通常CFに戻る
※高額療養費の自己負担上限を自動考慮

## 6. 計算ロジック

### 6.1 共通計算部品（先に実装）

#### A. 高額療養費制度の自己負担上限
```
入力：年収、月次医療費、70歳未満/以上
処理：年収から区分（ア〜オ）を判定
出力：月次自己負担上限額、年間負担額、多数回該当時の上限
```

#### B. 障害年金計算
```
入力：障害等級（1/2/3級）、加入年金（厚年/基礎のみ）、標準報酬月額
処理：基礎年金 + 厚生年金報酬比例 × 等級係数
出力：年金月額、年額
```

#### C. 傷病手当金計算
```
入力：標準報酬月額、休職開始日
処理：標準報酬月額 × 2/3 × min(休職期間, 1年6ヶ月)
出力：月額、総受取額、支給期間
```

#### D. 介護費用目安
```
入力：要介護度、介護形態（在宅/施設）
処理：公的介護保険の自己負担（1〜3割）+ 自費部分
出力：月額介護費用、年額
```

### 6.2 シナリオ別計算

#### 死亡
- 既存 `contingency.js` のロジックを流用
- 遺族年金（基礎＋厚生）を自動計算
- 生命保険金を発生年に一時金として計上

#### 高度障害
- 本人収入を発生年以降ゼロに
- 障害年金1級を毎年適用
- 介護費用を毎年加算
- 高度障害保険金を発生年に一時金として計上

#### 就労不能
- 本人収入を指定割合/金額で減額（生涯）
- 発症後1年6ヶ月は傷病手当金
- その後は障害年金（等級別）
- 就業不能保険の月額給付を毎月加算
- 一時金は発生年に計上

#### 三大疾病
- 休職期間中のみ収入減（傷病手当金2/3）
- 医療費支出は発生年に一時支出スパイク
- 高額療養費で自己負担に上限
- 疾病保険一時金を発生年に計上
- 復職後は通常CFに戻る

## 7. 出力・表示

### 7.1 リアルタイム更新
- **既存と同じ方式**（`live()` 関数を流用）
- 入力のたびに即座にCF表・サマリー・グラフが更新
- 新規の debounce 処理は導入しない

### 7.2 CF表の表示方式（案β）
- **サマリー**：常時表示（4シナリオ並列）
- **CF表本体**：タブで切替え
  - 通常 / 死亡 / 高度障害 / 就労不能 / 三大疾病
  - 選択中のシナリオだけタブ表示
- 既存のCF1/CF2タブとは別のエリアに配置

### 7.3 サマリー表の項目
| 行 | 内容 |
|---|---|
| 想定発生年齢 | 各シナリオで入力した年齢 |
| 発生直後の影響 | 発生年のCF差分 |
| 10年後の資産 | 通常比の資産残高 |
| 30年後の資産 | 通常比の資産残高 |
| 必要保障額 | シナリオ発生を乗り切るのに必要な金額 |
| 既保険カバー | 既契約保険で補填できる額 |
| **不足額** | **必要保障額 − 既保険カバー** |

### 7.4 資産推移グラフ
- Chart.js で4本の線を重ね描き
- 通常（青）＋ 選択中シナリオ（異なる色）
- 既存の graphs.js を拡張

## 9. 開発方針・段階

### 9.1 ブランチ戦略

```
main（メンバー使用中）
 ├─ 通常の小修正・バグ修正（継続）
 └─ feature/contingency-v2 ← ここで長期開発
     ├─ 定期的にmainから変更を取り込み
     └─ 完成後、動作確認してmainへマージ
```

### 9.2 段階的実装（フェーズ）

#### フェーズ0：**準備**

## 11. 確定事項サマリー

## 付録：用語集

| 用語 | 意味 |
|---|---|
| 標準報酬月額 | 社会保険料算出のベースとなる月給の区分額 |
| 報酬比例部分 | 厚生年金の保険料納付実績に応じた年金額 |
| 多数回該当 | 高額療養費で12ヶ月に3回以上該当した場合の特例 |
| 高度障害 | 生命保険約款上の重度障害状態（死亡保険金と同額給付） |
| 障害等級 | 1級（要常時介護）〜3級（軽度障害） |
| 傷病手当金 | 健康保険の休業補償（標準報酬×2/3） |
| 遺族基礎年金 | 国民年金の遺族給付（子供がいる場合） |
| 遺族厚生年金 | 厚生年金の遺族給付（報酬比例部分の3/4） |
| 要介護度 | 介護保険の認定区分（要支援1〜要介護5） |
